Package dedicated to all user interfaces that deal with glyphs. There are only a few top-level classes, each one using dedicated companion classes:
  • SymbolsEditor
    • SymbolGlyphBoard
    • EvaluationBoard
    • GlyphBoard
    • ShapeFocusBoard
  • GlyphVerifier
    • GlyphBrowser
  • GlyphTrainer
    • SelectionPanel
    • TrainingPanel
    • NetworkPanel
    • ValidationPanel
  • GlyphLagView
  • GlyphMenu
  • GlyphRepository
  • SpinnerGlyphModel
  • ShapeColorChooser
