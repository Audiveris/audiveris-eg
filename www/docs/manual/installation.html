Audiveris Installation Manual

Downloading Audiveris software

The latest version is always available on the Audiveris download area on Kenai site.

There is now one single distribution per release which, besides the binary components and user manuals, provides source code, JavaDoc, build procedures, training material, etc. Example: audiveris-3.3-full-20100519.zip

Just download the latest distribution (about 30MB) and expand this archive into some local directory on your computer.

System Requirements

  • OS: Audiveris is essentially a Java application, and as such should be usable on a wide variety of platforms. It is today being developed under Windows and Linux. It has run on Mac OS for some time, but the connection to Tesseract is not yet implemented for this platform. Since release 3.2a, Audiveris uses Google Tesseract OCR for the translation of textual glyphs. Tesseract was developed in C++ and is OS-dependent.
  • Java: Java 6 is highly recommended since it already contains the JAXB 2 component which is needed for XML persistency. If you cannot use Java 6, you can use Java 5 but you will have to download JAXB 2 from the JAXB web site and install it on your own.

Companion tools

Tesseract OCR

Tesseract OCR is a C++ program and Audiveris is a Java application. For the time being, the JNI connection between Audiveris and Tesseract has been implemented for Windows and Linux only.

  • For Windows environment, the Tesseract "tessdll.dll" component is used, via the Tesjeract Java companion. You still need the Microsoft C/C++ runtime, that you can install by running the vcredist_x86.exe installer downloadable from Microsoft Visual C++ 2008 SP1 Redistributable Package (x86). Note that uou no longer need to define the TESSDATA_PREFIX variable.
  • For Linux environment, download the tesjeract source from http://code.google.com/p/tesjeract/ and follow the provided instructions to build this interface.

Other components

All the other components need no special installation procedure, since their jar files are already provided in the dist/lib folder of the Audiveris installation directory and directly referenced from the manifest of the Audiveris jar file. See the Implementation manual if you need more detailed information on these components.
